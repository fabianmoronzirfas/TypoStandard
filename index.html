TypoStandard Workshop
by fabiantheblind

Some javascripts for InDesign for use @ the TypoStandard Scripting Workshop on friday the sixth may twotousendeleven

Install

let me google that for you

Run

Some of the scripts build a new document. Most of them use the first textframe an the first page of the active document.
You can:
- create a document and add a textframe with some text of your choice
or
- you run "buildDocument.jsx". This builds a document with a textframe and some text for you
